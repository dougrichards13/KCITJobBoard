# Job Board Quick Start Guide for HR

## Accessing the System

1. Open your web browser
2. Go to: **http://localhost:8001/admin**
3. Log in with your username and password

## Creating a Job Posting

1. Click **"Job Postings"** on the left sidebar
2. Click **"ADD JOB POSTING"** button (top right)
3. Fill in the form:
   - **Title**: Job position name (e.g., "Loan Officer")
   - **Location**: Where the job is located (e.g., "Remote", "Hybrid - Central Office")
   - **Description**: Full job description (what the role entails)
   - **Requirements** (optional): Skills and qualifications needed
   - **Responsibilities** (optional): Main duties
   - **Salary Range** (optional): e.g., "$50k-$70k"
4. Set **Status**:
   - **Draft**: You're still working on it (not visible to public)
   - **Active**: Ready to accept applications (visible on careers page)
   - **Closed**: No longer accepting applications
5. Click **"SAVE"**

## Viewing Applications

1. Click **"Job Applications"** on the left sidebar
2. You'll see a list of all applications

### Filter Applications
- Use the filters on the right side:
  - By Status (New, Under Review, Interview, etc.)
  - By Job Position
  - By Date

### Review an Application
1. Click on the applicant's name
2. You'll see:
   - Contact information (email, phone)
   - Cover letter (if provided)
   - Resume download button
   - LinkedIn profile (if provided)
3. Add **Internal Notes** if needed (candidate won't see these)
4. Change **Status** as you progress:
   - **New**: Just submitted
   - **Under Review**: You're reviewing it
   - **Interview Scheduled**: Moving forward
   - **Rejected**: Not a good fit
   - **Accepted**: Offer extended
5. Click **"SAVE"**

### Bulk Actions
- Check multiple applications
- Use **"Actions"** dropdown at the top to:
  - Mark as "Under Review"
  - Mark as "Interview Scheduled"
- Click **"Go"**

## Email Notifications

You'll receive an email notification whenever someone applies. The email contains:
- Candidate name and contact info
- Job they applied for
- Link to view the application in admin
- Resume download link

## Editing a Job Posting

1. Click **"Job Postings"**
2. Click on the job title
3. Make your changes
4. Click **"SAVE"**

## Closing a Job

1. Find the job in the list
2. Change **Status** to **"Closed"**
3. The job will no longer accept applications

## Tips

- **Draft Status**: Use this when creating a job posting to preview it before making it public
- **Search**: Use the search bar at the top to find specific candidates or jobs
- **Sort**: Click column headers to sort the list
- **Export**: You can export application lists if needed

## Need Help?

Contact the system administrator if you:
- Forget your password
- Need additional permissions
- Have technical issues
